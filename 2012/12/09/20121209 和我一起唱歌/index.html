和我一起唱歌

我下午在车站等车。非高峰时段，所等的公交久等不来，心烦意乱的踱步。忽然隐隐听到歌声传来，莫非附近有卡拉ok么，不对啊我想。于是我走到站牌后面，发现原来此处有一个地下通道。那我就懂了嘛，是地下通道常有的街头艺人在唱歌了。我对这些流行歌曲不太熟悉，只听歌词里有什么可惜不是你之类的。既然公车总也不来，等得烦躁，便去看看。这歌唱的不错，男声也好听，瞧瞧。

我走进地下通道，男孩在通道正中靠墙席地而坐，面前摆着吉他箱子，里面散放着零钱，人则只是专心的盯着正弹奏的吉他，嘴巴凑近麦克风自顾自的唱着，普通文艺青年所梦寐以求的特殊气质他很简单的就拥有了嘛。我既然是下来听唱歌的，那么也不必掩饰，于是我在他对岸的墙上靠着，双手抱臂舒服的站立。

尽管他唱的歌都是我从未听过的，但这歌声里感情充沛，正是唱歌最打动人的地方。我一边听一边沉思着，直到歌声和乐声忽然一齐停掉，我下意识的抬眼看对面，看到唱歌男孩也正看着我。很干净的一张脸，以及略带困惑的眼神。

“很好听。”又想了片刻我开口说道。

“谢谢。”男孩脸上的困惑和不解立刻被松了一口气的模样所取代了。

“不唱了吗？”我问。

男孩微笑一下，很甜的样子，然后又唱了起来。

男孩的歌声让人想起很多事。是啊，不知不觉，我也有了许多往事，就像这些情歌里唱的那样。

直到我起身走到男孩这边，俯身用手轻轻的摁住他的吉他，于是地下通道又一次静了下来。

“我可以点唱吗？”我微笑着问。

男孩歪起脑袋，像在想问题那样。

看到吉他箱子里的零钱，我突然想到了什么，于是从裤子荷包搜出全部三枚一元的硬币。

“我只是现在有一首非常想听的歌而已。”说着我把硬币轻轻放进吉他箱子。

“好吧，不过要我会唱才行啊。”

“《你不是真正的快乐》，没问题吧。”

“嗯，行。”男孩停顿了一下，又开口说道：“不过你要回答我一个问题。”

“什么？”

“为什么想听这首呢？”

是啊，为什么呢。可是哪有那么多为什么。也许只是因为不快乐？

“唱歌真好。”冒出来的只是这么个回答。

“我可以坐这里吗？”我不理会男孩匪夷所思的表情，指着他旁边的地上说道。

“嗯，坐。”

于是我也坐到了地上。打量着偶尔才会走过一两个的行人。

“人们唱歌，其实是在抒情。感情抒发得越充沛越淋漓，歌也就越打动人。所以，你的歌真的唱得挺棒的。”

“谢谢，不知道你唱的如何？”

“这个世界有会唱歌的人，就必定有不会唱歌的人。我属于后者。”

“所以你的意思是，你无法畅快的抒情？”

“也许吧。”我说：“换个方式来讲，这个世界有抒发感情顺畅无阻的人，就必定有感情抒发困难的人。只不过世界更多的看到前一种人，因为他们歌声好听。可是我知道，在离这种人很远的地方，还有另外一种人，他们感情压抑，似乎是天生的，不知为何就是无法顺畅的表达感情。我可以懂他们的困难和痛苦，所以比起会唱歌的人还是后者更让我感到安心。”

“嗯，我明白。”安静了一会，他又说：“天生顺畅的表达感情固然是好的，如果没办法做到，也许只是表明上帝在这方面对你加设了考验。假使你慢慢努力，或许也能变好的呢。”

我看着他。

“比如唱歌，就是最好的抒情练习也不一定。”他又说。

“抒情练习。”我逐字的重复出这个让我感觉很好的词。

“那，试着和我一起唱吧。”他用膝盖轻碰我的膝盖：“让你心中的感情，在唱歌中自然的流露出来就好，嗯。”

说完他就随性的拨动吉他唱了起来。

是啊，唱歌真好，听着重新流淌的歌声的我总是这么觉着。

和他经过麦克风扩音的声音比起来我的声音根本就会微不足道，于是我便也跟着他的弹奏轻声唱了起来，他转过头，给我送来鼓励的眼神。

“你不是真正的快乐 / 你的笑只是你穿的保护色 / 你决定不恨了 也决定不爱了 / 把你的灵魂关在永远锁上的躯壳”

“你值得真正的快乐 / 你应该脱下你穿的保护色 / 为什么失去了 还要被惩罚呢 / 能不能就让悲伤全部结束在此刻”
